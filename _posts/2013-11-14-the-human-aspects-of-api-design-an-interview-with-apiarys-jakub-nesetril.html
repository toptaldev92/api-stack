API Design and description is more than just a technical interface contract between machines.Apiarys Co-Founder and CEO Jakub Nesetril points out that the real consumer of an API description is the developer, with all the concerns of engagement, usability and communication that entails.We spoke with Jakub recently about Apiarys approach to API design and emerging API tools and workflows.InfoQ: Jakub, you recently presented at the API Strategy conference on a panel about best practices in API design.You mentioned there that there is no one way to build an API.
