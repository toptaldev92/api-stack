APIs That I Am Working To Certify

I am always working to make sure as many of the APIs in my stack are complete, but I only have so much time as a one man show. My goal is to get as many of them as close to complete as I can, and to help move things forward I am taking some of the more popular ones, and targeting separately to be certified.

In addition to investing in this work myself, I am also getting the assistance of API partners like Dream Factory, who are looking to get specific API definitions completed--here are the ones I am working on currently.

AngelList - Working to define all the endpoints for the startup, and investment platform API.
Alchemy - Working to define all the endpoints for the machine learning API.
Century Link - Working to define all the endpoints for the cloud computing API.
Docomo - Working to define all the endpoints for the telco, image, machine learning API.
Stipe - Working to define all the endpoints for the payment API.
Twilio - Working to define all the endpoints for the voice & messaging API.
Verizon - Working to define all the endpoints for their Internet of Things (IoT) API.

I've assigned each API its own Github repo to store the OpenAPI Specs, as well as indexing them using APIs.json, and facilitating the conversation using the Github issues.

I will work to certify each of these APIs as complete, and make sure each API has all the endpoints present, as well as the response, request, and security defined. If there is any API you'd like to see priotized, let me know.
